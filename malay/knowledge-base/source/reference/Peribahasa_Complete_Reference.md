# PSLE Malay Language — Peribahasa (Proverbs) Complete Reference
> For P6 PSLE 2026 preparation
> Peribahasa appears in Paper 2 Booklet A (typically 5 MCQ questions)
> Also valuable for Paper 1 Composition (karangan) — used correctly, adds marks

---

## How Peribahasa is Tested

**Paper 2 (Tatabahasa section):** 5 MCQ questions
- Given a context/situation → choose the correct peribahasa
- OR given a peribahasa → choose the correct meaning
- OR fill a blank with the correct peribahasa

**Paper 1 (Karangan):** Using peribahasa in composition
- 1–2 peribahasa used CORRECTLY can add marks
- Wrong usage REDUCES marks — so only use what you know well
- Use in context naturally — don't force it

## HOW TO USE PERIBAHASA IN KARANGAN

### ✅ Correct Usage — Natural, in-context

> *Ahmad membantu rakannya yang baru berpindah ke kawasan itu. Sebagai seorang jiran yang baik, Ahmad tahu bahawa "seperti aur dengan tebing", mereka harus saling membantu.*

**Why it works:** The proverb fits naturally into the narrative. It extends the meaning of the scene.

---

### ❌ Wrong Usage — Forced, out of context

> *Saya sangat gembira hari ini. "Berakit-rakit ke hulu, berenang-renang ke tepian." Saya juga suka bermain.*

**Why it fails:** The proverb is about hard work and sacrifice — completely unrelated to the sentences around it.

---

### Tips for Using Peribahasa in Composition
1. Only use proverbs you know well — wrong use = mark deduction
2. Use 1–2 per composition maximum
3. Introduce with: *"seperti peribahasa yang berbunyi..."* or embed naturally in narration
4. Make sure the context matches the proverb's meaning exactly
5. If unsure — skip it. A clean, proverb-free composition is safer
