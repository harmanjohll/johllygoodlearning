## What is Tatabahasa?

Tatabahasa is the grammar system of the Malay language. In PSLE Paper 2 Booklet A, approximately **40 marks** are dedicated to language usage (tatabahasa). This is the single biggest scoring opportunity in the entire exam.

The main areas tested are:
1. **Imbuhan** (Affixes)
2. **Golongan Kata** (Word Classes / Parts of Speech)
3. **Peribahasa** (Proverbs) — covered in separate file
4. **Frasa dan Ayat** (Phrases and Sentences)

---

## SECTION 1: IMBUHAN (Affixes)

Imbuhan are added to root words (kata dasar) to change meaning or grammatical function. This is the most heavily tested area in PSLE Malay grammar.

### Types of Imbuhan

| Type | Name | Example |
|---|---|---|
| Awalan (Prefix) | me-, ber-, pe-, ter-, ke-, di- | menulis, berlari, penulis, terjatuh |
| Akhiran (Suffix) | -kan, -an, -i | tuliskan, tulisan, menyakiti |
| Apitan (Circumfix) | me-...-kan, me-...-i, ke-...-an, pe-...-an | menurunkan, memahami, keindahan, pemahaman |

---

### 1.1 Awalan ME- (Active verb prefix)

**Function:** Forms active verbs (kata kerja aktif). The subject performs the action.

**Spelling rules:**

| Root word begins with... | ME- becomes... | Example |
|---|---|---|
| m, n, ng, ny | me- (unchanged) | me + masak = **memasak** |
| p | mem- (p drops) | me + potong = **memotong** |
| b | mem- | me + beli = **membeli** |
| t | men- (t drops) | me + tulis = **menulis** |
| d | men- | me + dorong = **mendorong** |
| c | men- | me + cari = **mencari** |
| j | men- | me + jual = **menjual** |
| k | meng- (k drops) | me + kira = **mengira** |
| g | meng- | me + gambar = **menggambar** |
| a, e, i, o, u | meng- | me + ambil = **mengambil** |
| s | meny- (s drops) | me + sapu = **menyapu** |
| l, r | me- | me + lari = **melari** (berlari more natural) |
| f | mem- | me + fikir = **memikirkan** |

**Common mistakes:**
- ❌ mukul → ✅ memukul
- ❌ baca → ✅ membaca
- ❌ tulis → ✅ menulis
- ❌ kumpul → ✅ mengumpul

---

### 1.2 Awalan BER- (Stative/intransitive verb prefix)

**Function:** Indicates a state, condition, activity, or possession. Often intransitive (no object).

**Rules:**
- BER- + most words → berlari, berjalan, bermain, berbicara
- BER- + root starting with 'r' → drops one 'r': ber + rasa = **berasa**
- BER- + root starting with 'ajar' (special case) → **belajar** (not 'berajar')

**Common BER- words (important for PSLE):**

| Word | Meaning |
|---|---|
| berlari | to run |
| berjalan | to walk |
| bermain | to play |
| berbicara | to speak/converse |
| bertanya | to ask |
| berkata | to say |
| berhenti | to stop |
| berenang | to swim |
| berfikir | to think |
| berharap | to hope |
| bersama | together |
| berterima kasih | to thank |
| bersungguh-sungguh | wholeheartedly |

---

### 1.3 Awalan PE- (Noun-forming prefix — person/agent)

**Function:** Creates nouns referring to a person who does something.

| Root | PE- word | Meaning |
|---|---|---|
| tulis | penulis | writer |
| masak | pemasak | cook |
| jual | penjual | seller |
| beli | pembeli | buyer |
| main | pemain | player |
| tolong | penolong | helper |

**Spelling rules follow same consonant changes as ME-**

---

### 1.4 Awalan TER- (Unintentional/superlative)

**Function:** Indicates:
- **Accidental action:** terjatuh (accidentally fell), terlupa (accidentally forgot)
- **Superlative:** terbesar (biggest), terpandai (cleverest), tercantik (most beautiful)
- **Completed state:** terbuka (open), tertutup (closed)

| Word | Meaning |
|---|---|
| terjatuh | to fall (accidentally) |
| terlupa | to forget (accidentally) |
| tersilap | to make a mistake (accidentally) |
| terbesar | the biggest |
| tercantik | the most beautiful |
| terpandai | the cleverest |
| terbuka | open (state) |

---

### 1.5 Akhiran -KAN (Causative suffix)

**Function:** Makes a verb transitive (action done FOR or TO someone/something)

| Word | With -KAN | Meaning |
|---|---|---|
| beli | belikan | buy (for someone) |
| ambil | ambilkan | take/fetch (for someone) |
| tulis | tuliskan | write (something) |
| buka | bukakan | open (for someone) |
| cerita | ceritakan | tell (a story) |
| jaga | jagakan | keep/guard (something) |

---

### 1.6 Akhiran -AN (Creates nouns)

**Function:** Turns verbs or adjectives into nouns

| Root | With -AN | Meaning |
|---|---|---|
| tulis | tulisan | writing |
| lukis | lukisan | drawing/painting |
| makan | makanan | food |
| minum | minuman | drink |
| jalan | jalanan | road/path |
| kerja | kerjaan | work/task |
| main | mainan | toy |
| pelajar | pelajaran | lesson |

---

### 1.7 Apitan KE-...-AN (Creates nouns — state or condition)

**Function:** Creates abstract nouns expressing a state, quality, or condition

| Root | KE-...-AN word | Meaning |
|---|---|---|
| indah | keindahan | beauty |
| selamat | keselamatan | safety |
| bersih | kebersihan | cleanliness |
| baik | kebaikan | goodness/kindness |
| susah | kesusahan | difficulty |
| gembira | kegembiraan | happiness/joy |
| sedih | kesedihan | sadness |
| berani | keberanian | bravery |
| bijak | kebijaksanaan | wisdom |
| maju | kemajuan | progress |
| hilang | kehilangan | loss |
| tiba | ketibaan | arrival |

---

### 1.8 Apitan PE-...-AN (Creates nouns — process/result)

| Root | PE-...-AN word | Meaning |
|---|---|---|
| belajar | pembelajaran | learning |
| fikir | pemikiran | thinking/thought |
| buat | perbuatan | action/deed |
| jalan | perjalanan | journey |
| tumbuh | pertumbuhan | growth |

| sedia | penyediaan | preparation |

## SECTION 2: GOLONGAN KATA (Word Classes)

### 2.1 Kata Nama (Nouns)

Words that name people, places, things, or abstract concepts.

**Types:**
- **Kata nama am (common nouns):** buku, meja, pokok, sekolah
- **Kata nama khas (proper nouns):** Ahmad, Singapura, Sungai Muar
- **Kata nama abstrak (abstract nouns):** kasih sayang, keberanian, kemajuan

---

### 2.2 Kata Kerja (Verbs)

Words that describe actions or states.

**Types:**
- **Kata kerja transitif (transitive):** requires an object — membaca buku, memukul bola
- **Kata kerja tak transitif (intransitive):** no object — berlari, berdiri, duduk

**Common kata kerja for PSLE:**

| Kata Kerja | Meaning |
|---|---|
| bergegas | to hurry |
| menjerit | to scream |
| meraung | to wail/sob |
| terpinga-pinga | to be dazed/confused |
| memeriksa | to examine/check |
| merasa | to feel |
| melambai | to wave |
| memeluk | to hug |
| menghiburkan | to comfort/entertain |

---

### 2.3 Kata Adjektif (Adjectives)

Words that describe nouns.

**Types:**
- **Kata adjektif sifatan:** describes quality — cantik, besar, pantas
- **Kata adjektif bilangan:** describes quantity — beberapa, semua, banyak
- **Kata adjektif cara:** describes manner — perlahan-lahan

**Degrees of comparison:**

| Degree | Form | Example |
|---|---|---|
| Positive | Adjective alone | cantik (beautiful) |
| Comparative | lebih + adjective | lebih cantik (more beautiful) |
| Superlative | paling + adjective | paling cantik (most beautiful) |
| Also superlative | ter- + adjective | tercantik |
| Equative | se- + adjective | secantik (as beautiful as) |

---

### 2.4 Kata Tugas (Function Words)

Words that connect, modify, or show relationships.

**Sub-types:**

| Sub-type | Examples | Usage |
|---|---|---|
| Kata hubung (conjunctions) | dan, atau, tetapi, kerana, supaya, walaupun, apabila | Connect clauses |
| Kata seru (interjections) | Aduh!, Wah!, Eh!, Syabas! | Express emotion |
| Kata tanya (interrogatives) | apa, siapa, bila, di mana, bagaimana, mengapa | Form questions |
| Kata ganti nama (pronouns) | saya, awak, dia, mereka, kami, kita | Replace nouns |
| Kata bilangan (numbers/quantifiers) | satu, dua, beberapa, semua, ramai | Quantity |
| Kata arah (directional) | ke, dari, kepada, daripada | Direction/source |
| Kata sendi nama (prepositions) | di, ke, dari, dengan, untuk, oleh | Location/relation |

## SECTION 3: PEMBENTUKAN AYAT (Sentence Structure)

### Basic Sentence Pattern (FIK)
**Frasa Nama + Frasa Kerja** = Subject + Predicate

Example: **Ahmad** (FN) + **berlari dengan pantas** (FK)

### Common Sentence Types

| Type | Structure | Example |
|---|---|---|
| Ayat penyata (statement) | Subject + Predicate | Dia pergi ke kedai. |
| Ayat tanya (question) | Question word + Sentence | Di manakah kamu tinggal? |
| Ayat perintah (command) | Verb + Object | Tutup pintu itu! |
| Ayat seru (exclamation) | Interjection + Sentence | Alangkah indahnya pemandangan ini! |

### Kata Hubung (Conjunctions) — Important for Comprehension & Composition

| Kata Hubung | Meaning | Example |
|---|---|---|
| dan | and | Ali dan Abu bermain. |
| atau | or | Kamu mahu teh atau kopi? |
| tetapi / namun | but / however | Dia rajin tetapi pemalu. |
| kerana | because | Dia menangis kerana sedih. |
| supaya / agar | so that | Belajarlah supaya berjaya. |
| walaupun / meskipun | although | Walaupun penat, dia terus berusaha. |
| apabila / ketika | when | Apabila hujan turun, dia pulang. |
| setelah / selepas | after | Setelah makan, dia tidur. |
| sebelum | before | Sebelum tidur, gosok gigi. |
| jika / jikalau / kalau | if | Jika rajin, pasti berjaya. |
| sambil | while (simultaneously) | Dia menyanyi sambil memasak. |
| lalu / kemudian | then / afterwards | Dia jatuh lalu menangis. |
| sehingga | until / so much that | Dia berlari sehingga penat. |

## SECTION 4: COMMON ERRORS TO AVOID

### Affix Errors (Most Common in PSLE)

| Wrong | Correct | Rule |
|---|---|---|
| mukul | memukul | me- + p/b words → mem- |
| baca | membaca | me- + b words → mem- |
| tulis | menulis | me- + t words → men- |
| kumpul | mengumpul | me- + k words → meng- |
| sapu | sapu ✅ or menyapu | me- + s words → meny- |
| cerita | menceritakan (if used as verb) | don't forget me- for verbs |
| pergi (verb) | no affix needed | pergi is already a complete verb |

### Subject-Verb Agreement

Malay does not conjugate verbs by person — BUT the form must match the intended meaning:
- ❌ "Dia ber-lari ke sana" (ber-lari is rarely used alone; berlari is correct)
- ✅ "Dia berlari ke sana"

### Kata Ganti Nama (Pronoun) Usage

| Pronoun | Usage |
|---|---|
| saya | formal "I" |
| aku | informal "I" (for composition: use based on character) |
| awak | formal "you" (to someone of equal or lower status) |
| kamu | informal "you" |
| anda | formal "you" (public/impersonal) |
| dia | he/she (neutral) |
| beliau | he/she (respectful — for elders, important people) |
| mereka | they |
| kami | we (exclusive — not including listener) |
| kita | we (inclusive — including listener) |

- [ ] Distinguish kami vs. kita, dia vs. beliau
