## GRAMMAR SCHEMA: IMBUHAN

```
IMBUHAN (Affixes)
├── Awalan (Prefixes)
│   ├── me-    → Active verb    → memasak, menulis, mengambil
│   ├── ber-   → Intransitive   → berlari, bermain, berbicara
│   ├── pe-    → Agent noun     → penulis, pemasak, pemain
│   ├── ter-   → Accidental/Superlative → terjatuh, terbesar
│   ├── ke-    → Ordinal/Abstract → ketiga, kelima
│   └── di-    → Passive verb   → dibaca, ditulis, diambil
│
├── Akhiran (Suffixes)
│   ├── -kan   → Causative/Transitive → tuliskan, ambilkan
│   └── -an    → Noun             → tulisan, makanan
│
└── Apitan (Circumfixes)
    ├── me-...-kan  → Causative active  → menurunkan, membesarkan
    ├── me-...-i    → Repetitive/Intensive → memahami, mengilai
    ├── ke-...-an   → Abstract noun    → keindahan, kesedihan
    ├── pe-...-an   → Process noun     → pembelajaran, perjalanan
    └── ber-...-an  → Reciprocal       → bertemuan, bercakapan
```

## GRAMMAR SCHEMA: GOLONGAN KATA

```
GOLONGAN KATA (Word Classes)
│
├── Kata Nama (Nouns)
│   ├── Kata Nama Am (Common Nouns): buku, pokok, sekolah
│   ├── Kata Nama Khas (Proper Nouns): Ahmad, Singapura
│   └── Kata Nama Abstrak (Abstract): kasih, keberanian
│
├── Kata Kerja (Verbs)
│   ├── Kata Kerja Transitif: requires object → membaca, menulis
│   └── Kata Kerja Tak Transitif: no object → berlari, duduk
│
├── Kata Adjektif (Adjectives)
│   ├── Sifatan: cantik, besar, pantas
│   ├── Bilangan: beberapa, semua, ramai
│   └── Cara: perlahan-lahan, tergesa-gesa
│
└── Kata Tugas (Function Words)
    ├── Kata Hubung: dan, atau, tetapi, kerana, supaya
    ├── Kata Seru: Aduh!, Wah!, Syabas!
    ├── Kata Tanya: apa, siapa, bila, di mana
    ├── Kata Ganti Nama: saya, dia, mereka, kami, kita
    ├── Kata Sendi Nama: di, ke, dari, dengan, untuk
    └── Kata Bilangan: satu, dua, beberapa, ramai
```

## KARANGAN SCHEMA

```
KARANGAN (Composition)
│
├── Karangan Bertajuk (Topic-Based)
│   ├── Planning Required: Who, Where, When, What, Why, How
│   ├── Structure: 5 paragraphs (Intro → Rising → Climax → Falling → Conclusion)
│   └── Key techniques: Atmosphere opener, rich description, varied dialogue, peribahasa
│
├── Karangan Bergambar (Picture-Based)
│   ├── 3–4 pictures provided
│   ├── 1 paragraph per picture
│   ├── Must: add story before pic 1 and after last pic
│   └── Key techniques: Smooth transitions, emotion-rich narration
│
└── Marking Criteria
    ├── Bahasa (Language): grammar, vocab, affixes, spelling    [15 marks]
    ├── Isi (Content): plot logic, creativity, relevance        [15 marks]
    └── Gaya (Style): peribahasa, dialogue, atmosphere         [10 marks]
```

## ORAL SCHEMA

```
ORAL COMMUNICATION (Lisan)
│
├── Membaca Kuat (Reading Aloud)
│   ├── Pronunciation (Sebutan): Malay phonics, stress
│   ├── Fluency (Kelancaran): no excessive pauses
│   ├── Expression (Ekspresi): appropriate intonation
│   └── Accuracy: no added/missing words
│
└── Perbualan Berasaskan Rangsangan (Stimulus Conversation)
    ├── Input: Short video clip
    ├── Response Framework: OREO (Opinion, Reason, Example, Opinion)
    ├── Sentence Starters: Pada pendapat saya... / Ini kerana... / Contohnya...
    └── Common Themes:
        ├── Environment / Alam sekitar
        ├── Health / Kesihatan
        ├── Family / Keluarga
        ├── School / Sekolah
        ├── Values / Nilai murni
        └── Singapore Identity
```

## COMPREHENSION SCHEMA

```
COMPREHENSION (Kefahaman)
│
├── Text Types
│   ├── Naratif (story)
│   ├── Ekspositori (informational)
│   ├── Prosedural (instructions)
│   └── Persuasif (argument)
│
├── Question Types
│   ├── Tersurat (Literal): answer in text
│   ├── Tersirat (Inferential): read between lines
│   ├── Vocabulary-in-Context: meaning of word/phrase
│   └── Language Use: why writer chose specific language
│
└── Answer Framework
    ├── Quote from passage
    ├── Link to question
    └── Explain / Justify
```

## PERIBAHASA SCHEMA (for digital practice)

```
PERIBAHASA
├── By Theme
│   ├── Relationships: bagai aur dengan tebing, seperti isi dengan kuku
│   ├── Hard Work: rajin pangkal pandai, berakit-rakit ke hulu
│   ├── Wisdom: biar lambat asal selamat, terlajak perahu boleh diundur
│   ├── Consequences: seperti kacang lupakan kulit, menepuk air di dulang
│   ├── Happiness: seperti kera mendapat bunga, bagai mendapat durian runtuh
│   ├── Hardship: berat mata memandang, sudah jatuh ditimpa tangga
│   └── Environment: air yang tenang jangan disangka tiada buaya
│
└── Question Formats in Exam
    ├── Match situation → peribahasa
    ├── Given peribahasa → explain meaning
    ├── Fill blank in sentence with correct peribahasa
    └── Choose peribahasa that fits a context (MCQ)
```

## VOCABULARY SCHEMA (by PSLE relevance)

```
VOCABULARY
├── Emotions (Emosi): gembira, sedih, terharu, cemas, bangga, lega
├── Movement (Pergerakan): berlari, bergegas, menghampiri, tersungkur
├── Character traits (Sifat): rajin, berani, jujur, penyayang, bijaksana
├── Setting (Latar): pemandangan indah, suasana mencengkam, penuh sesak
├── Community (Komuniti): gotong royong, perpaduan, saling menghormati
├── Linking words (Kata penghubung): selain itu, namun, oleh itu, tiba-tiba
└── Penjodoh Bilangan: orang, ekor, buah, helai, batang, biji
```

## FOR CLAUDE CODE — PRACTICE QUESTION TYPES

The following question types should be buildable from the knowledge base:

### Paper 2 Tatabahasa Practice
```json
{
  "type": "imbuhan_mcq",
  "question": "Pilih imbuhan yang betul: Ahmad ___ adiknya ke sekolah. (bawa)",
  "options": ["membawa", "terbawa", "dibawa", "perbawa"],
  "answer": "membawa",
  "explanation": "ME- + bawa = membawa (active verb: Ahmad is the subject performing the action)"
}
```

### Peribahasa Practice
```json
{
  "type": "peribahasa_match",
  "scenario": "Zaid dan Hafiz selalu membantu antara satu sama lain. Mereka tidak pernah berpisah.",
  "options": [
    "Seperti isi dengan kuku",
    "Hangat-hangat tahi ayam",
    "Seperti kacang lupakan kulit",
    "Bagai api dengan air"
  ],
  "answer": "Seperti isi dengan kuku",
  "explanation": "Seperti isi dengan kuku = inseparable, close friends who always support each other"
}
```
