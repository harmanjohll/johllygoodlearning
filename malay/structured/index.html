📋 Karangan & Kefahaman Builder

Latih karangan atau kefahaman dengan dua mod: Pemeriksa (markah) atau Coach (probe sahaja)
⚠️ API key Gemini diperlukan untuk mod Pemeriksa dan Coach. Tanpa key, anda masih boleh menggunakan halaman ini untuk berlatih menulis dengan rangka, tetapi maklum balas AI tidak akan tersedia.
Mod Pemeriksa menilai karangan/jawapan anda mengikut rubrik PSLE: kosa kata, tatabahasa, struktur, kemudian beri band anggaran dan ulasan ringkas.
Mod Coach mengajukan SATU soalan probe pada satu masa. Tidak mendedahkan jawapan model sehingga anda minta. Lebih sesuai untuk konsolidasi mendalam.
