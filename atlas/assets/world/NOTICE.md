# World map asset

`world.svg` is **Simple SVG World Map** by Fritz Lekschas
(https://github.com/flekschas/simple-world-map), a lightweight world map whose
`<path>` / `<g>` elements are annotated by **ISO 3166-1 alpha-2** country code —
the same key used to join the country dataset in `../../data/countries.js`.

It is licensed under **Creative Commons Attribution-ShareAlike 3.0 Unported
(CC BY-SA 3.0)**: https://creativecommons.org/licenses/by-sa/3.0/

It is bundled locally so Family Atlas works fully offline. The map is used, with
attribution, for the "Map Locate" and "Name the Shape" games and the passport
"Where you've been" map. A visible credit is shown beneath the map in the app.

The map covers 174 of the 194 countries in the dataset; the ~20 micro-states it
omits (e.g. Vatican, Monaco, Tuvalu) are excluded from the two map games but
remain fully playable in the flag, capital and country-name games.
