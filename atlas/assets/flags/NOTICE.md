# Flag assets

The SVG flags in this folder are from **flag-icons** by Panayiotis Lipiridis
(https://github.com/lipis/flag-icons), released under the **MIT License**.

They are bundled locally (rather than loaded from a CDN) so Family Atlas works
fully offline and so flags render correctly on every platform — including
Windows, where Unicode flag emoji fall back to two-letter codes.

Files are named by ISO 3166-1 alpha-2 code (lowercase), e.g. `fr.svg`, which is
the same key used to join the country dataset in `../../data/countries.js`.
